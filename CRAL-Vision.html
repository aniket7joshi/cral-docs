Basic requirement for the networks is that all codes should be compatible with tf.2.2 and above

Classification Networks

Development of classification networks for CRAL should follow the following protocol.

  • Find a Keras (tf.keras) implementation of the network,
  • Quick test on some dataset(less epochs)
  • Add it to cral.models.classification module
  • Add the keras definition in a .py file and put it cral→models→classification
  • Add alias of it int cral→models→__init__.py
  • Check that the implementation is working properly
  • Add to classification pipe
