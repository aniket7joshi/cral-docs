Basic requirement for the networks is that all codes should be compatible with tf.2.2 and above

Classification Networks

Development of classification networks for CRAL should follow the following protocol.

  1. Find a Keras (tf.keras) implementation of the network,
  2. Quick test on some dataset(less epochs)
  3. Add it to cral.models.classification module
  4. Add the keras definition in a .py file and put it cral→models→classification
  5. Add alias of it int cral→models→__init__.py
  6. Check that the implementation is working properly
  7. Add to classification pipe

Below are the list of classification networks for CRAL

  • DenseNet (121, 169, 201)
  • InceptionResNetV2
  • InceptionV3
  • MobileNet
  • MobileNetV2
  • NASNET (Large, Mobile)
  • ResNet (50, 101, 152)
  • ResNetV2  (50, 101, 152)
  • Xception
  • VGG (16,19)
  • EfficientNet (B0-B7)
  • Darknet53
  • DetNet
